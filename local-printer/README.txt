
============================================================
              AUTO PRINTER INSTALLATION GUIDE
============================================================

To move this program to another computer:

1. Install Node.js:
   - Download "Node.js (LTS version)" from: https://nodejs.org/
   - Install it on the new computer.

2. Copy Folder:
   - Copy this entire "local-printer" folder to the new computer (e.g., to Desktop).

3. Setup (Only once):
   - Open this folder on the new computer.
   - Right-click inside the empty space and select "Open Terminal" (or PowerShell).
   - Type this command and press Enter:
     npm install

4. Run:
   - Double-click "start-printer.bat".
   - Go to http://localhost:4000 to set the new printer name.
